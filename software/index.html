Installation

You can install any of my packages via the following code in R:

if (!require("devtools")) {
  install.packages("devtools")
}
devtools::install_github("mkanai/{package_name}")

deFinetti

GitHub

A package for plotting a de Finetti diagram and distributions of F-statistics of genotypes.

ccmatch

GitHub

A package for optimal matching of cases to controls using network flow theory.

Reference

  • Rosenbaum, P. R. Optimal matching for observational studies. Journal of the American Statistical Association 84, 1024–1032 (1989). doi:10.1080/01621459.1989.10478868

WRShd

GitHub

This package provides C++ sub-routines for the Harrell-Davis estimate functions from R. R. Wilcox' robust statistics package WRS, which substantially speed up the original implementation in R.
